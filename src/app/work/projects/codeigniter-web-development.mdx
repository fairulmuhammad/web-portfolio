## MVC Architecture Implementation

### Model Layer
```php
class Student_model extends CI_Model {
    public function get_students($limit = FALSE, $offset = FALSE) {
        if ($limit) {
            $this->db->limit($limit, $offset);
        }
        $query = $this->db->get('students');
        return $query->result_array();
    }
}
```

## Database Design

### Key Concepts Learned
- **Normalization**: Proper database structure to eliminate redundancy
- **Relationships**: One-to-many and many-to-many table relationships
- **Indexing**: Optimizing queries for better performance
- **Constraints**: Ensuring data integrity with foreign keys

### Sample Database Schema
```sql
CREATE TABLE students (
    id INT PRIMARY KEY AUTO_INCREMENT,
    student_id VARCHAR(20) UNIQUE NOT NULL,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE courses (
    id INT PRIMARY KEY AUTO_INCREMENT,
    course_code VARCHAR(10) UNIQUE NOT NULL,
    course_name VARCHAR(100) NOT NULL,
    credits INT NOT NULL
);
```
